邊境據點 Web 版：遊玩內容與關卡要素整理

這款遊戲是俯視角的生存建造遊戲，結合了大地圖探索、資源採集、基地防守、即時戰鬥、職業能力、區域事件與主線關卡推進。

一、核心玩法

1. 生存與資源
- 玩家需要在大地圖上移動，收集木材、石材、纖維、水晶等資源。
- 同時要注意自己的生命、飽食，以及基地核心穩定度。
- 飽食過低會持續扣血，核心穩定度歸零則遊戲失敗。

2. 建造與防守
- 玩家可以消耗資源建造不同防禦建築。
- 可建造項目包含：
  圍牆：基礎阻擋敵人。
  砲塔：自動攻擊接近的敵人。
  營地：靠近時可幫玩家回血。
  黑暗小法師塔：可遠距毒傷敵人，並召喚採集小兵。
  房子：可召喚菁英士兵協助戰鬥。
- 敵人會持續一波一波進攻基地，所以前中期的建設節奏很重要。

3. 戰鬥方式
- 左鍵：射擊
- 右鍵：丟毒藥水
- Space：近戰攻擊
- T：丟炸彈
- E：互動、採集、傳送、神社觸發等
- Q：切換建造模式
- 1 到 5：切換建築種類

4. 其他生存行動
- H：釣魚
- R：吃食物
- ESC：暫停
- F5 / F9：快速存檔 / 讀檔
- O / I：匯出 / 載入文字存檔

二、職業系統

1. 無業遊民
- 沒有特殊能力。

2. 吸血鬼
- 每 3 次命中可吸血回血。
- 可用 J 快速回到核心。

3. 伐木工
- 採集效率特別高。
- 很適合前期快速發育和蓋基地。

4. 漁夫
- 可以放船、上船。
- 船隻會自動開火。

5. 外星人
- 可用 J 發動八連砲。
- 可用 K 召喚小兵。

6. 菁英
- 承受傷害較低。
- 毒藥水傷害更強。

三、地圖與區域要素

1. 基地主區
- 遊戲主要防守核心。
- 商店與建築區也集中在這附近。

2. 神社
- 可召喚黑暗大法師。
- 完成條件後可每日領取永久紅寶石。

3. 綠國
- 地圖左上角的重要勢力區域。
- 內有房屋、軍營、議事廳、守軍、法師與村民。
- 綠國生物會成長、補兵，數量過多時還會主動集結進攻玩家核心。
- 若你滅掉整個綠國，會再觸發綠國大魔王戰。
- 打贏後綠國會變成你的地盤。

4. 損壞的村莊
- 第三關主要舞台。
- 進入後會觸發山匪與黑暗大法師事件。
- 村內藍基地被摧毀後，會掉落綠寶石。

5. 邪教堡壘
- 可由傳送門進入。
- 區域危險度更高。

6. 黑色禁地
- 最終 Boss 戰場。
- 玩家需要到此主動召喚灰暗魔王。

四、關卡流程

第一關：採集與神社
- 目標是收集足夠木材與石材。
- 同時前往神社召喚黑暗大法師，並累積指定擊殺數。
- 完成後進入第二關。

第二關：建立防線
- 目標是把基地建到一定規模。
- 需要足夠數量的砲塔、營地、黑暗小法師塔。
- 還要累積一定的怪物擊殺數。
- 這一關重點是經營與防守節奏。

第三關：損壞的村莊
- 進入村莊後會出現山匪與黑暗大法師。
- 需要擊殺足夠敵人、摧毀藍基地。
- 撿回綠寶石並送回核心，才能完成關卡。
- 完成後會獲得新的黑暗小法師塔支援。

第四關：灰暗魔王
- 前往黑色禁地。
- 按下互動鍵主動召喚最終 Boss。
- 擊敗灰暗魔王即通關。

五、特殊事件與系統

1. 神社紅寶石系統
- 打完指定數量黑暗大法師後，神社每天可領一次永久紅寶石。

2. 綠國戰爭系統
- 綠國不是單純背景，而是會持續生產單位。
- 當綠國兵力膨脹到一定程度，會有大規模突襲。

3. 村莊綠寶石事件
- 綠寶石是第三關推進關鍵。
- 沒有把它送回核心，就無法完成主線。

4. 多種友軍支援
- 砲塔可自動射擊。
- 黑暗小法師塔可上毒與召喚採集單位。
- 房子可生出菁英兵。
- 外星人與漁夫職業也能額外擴張戰力。

六、整體遊玩感受

這款遊戲不是單純打怪，也不是單純蓋塔，而是把探索、採集、據點建設、守城、主線任務、區域事件和 Boss 戰混在一起。前期偏生存與發育，中期偏防線經營與地圖事件，後期則進入區域征服與最終 Boss 戰。
